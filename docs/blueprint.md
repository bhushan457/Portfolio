# **App Name**: Bhushan Sonawane: Developer Portfolio

## Core Features:

- Responsive Navigation System: Implement a sticky navigation bar with links to Home, About, Skills, Projects, and Contact sections, ensuring smooth scrolling and hover animations on menu items across all device sizes.
- Animated Hero Section: Prominently display 'Bhushan Sonawane - Software Developer' with an animated text reveal, a tagline, call-to-action buttons (View Projects, GitHub, LinkedIn, Contact Me), and subtle background animation (particles or gradient).
- Dynamic Content Showcase: Present detailed About Me, Skills (with animated cards), and Projects (grid layout with interactive cards, descriptions, tech stack, and external links) sections, all featuring smooth fade-in and scroll animations.
- Functional Contact Form: Provide a modern contact form with fields for Name, Email, and Message, including input validation, animated input fields, and a submission mechanism using EmailJS or similar frontend solution to send emails, along with a success message.
- Unified Animation Engine: Utilize Framer Motion for comprehensive animations including page load, scroll reveals, various hover effects (buttons, cards), scaling, and smooth transitions throughout the entire application.
- Modern & Responsive UI/UX: Ensure the application adheres to a clean, minimal, and professional dark theme design with modern typography, optimized spacing, responsive layouts, and smooth scrolling for a seamless user experience across desktop, tablet, and mobile devices.
- Modular Code Architecture: Organize the codebase with reusable React components and a clear folder structure for maintainability and scalability.

## Style Guidelines:

- Primary color: A vibrant purple (#A667E4) to serve as a distinctive accent against the dark theme, symbolizing creativity and technological expertise. Calculated from HSL(270, 70%, 65%).
- Background color: A very dark gray with a subtle purple tint (#1A161D) for a sophisticated, modern dark theme base. Calculated from HSL(270, 15%, 10%).
- Accent color: A vivid electric blue (#1A1AE6) to provide dynamic contrast and highlight interactive elements or call-to-action buttons, drawing attention without being overwhelming. Calculated from HSL(240, 80%, 50%).
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-inspired, and impactful display of titles and the developer's name.
- Body font: 'Inter' (sans-serif) for highly legible and neutral body text, ensuring readability of descriptions and content.
- Use modern, outlined icons for social links (GitHub, LinkedIn, Email) and clear, distinct icons for skill categories to maintain a professional and minimal aesthetic. Incorporate icons from React Icons.
- Adopt a clean, spacious, and minimalist layout with balanced whitespace to enhance readability and visual appeal, featuring responsive grid systems for projects and skill cards. Integrate glassmorphism effects subtly on key UI elements like cards or buttons for a premium feel.
- Implement smooth and intentional animations using Framer Motion across the entire site, including graceful page entry, reveal animations on scroll for content sections, subtle hover effects on interactive elements, and dynamic text animation in the hero section to create an engaging user experience.
